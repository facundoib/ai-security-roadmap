# AI Security Roadmap

A curated 8-week roadmap for software and security engineers pivoting into AI Security. Published at [fibarzabal.com](https://fibarzabal.com).

Only the resources worth your time. No tutorials, no filler. New bimesters are unlocked as the author completes them.

## Stack

- [Astro](https://astro.build) 5.x — static site generator
- Plain CSS, system font stack, dark mode only
- No backend, no tracking, no cookies
- Progress stored client-side in `localStorage`

## Local development

```bash
npm install
npm run dev
```

Build:

```bash
npm run build
```

Output is written to `dist/`.

## Deploy

Auto-deployed by Netlify on push to `main`. Build command: `npm run build`. Publish dir: `dist`.
